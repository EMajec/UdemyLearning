.text() allows us to get the text, or change text
.html() is the equivalent of innerHTML in javascript
.attr("element","attribute") retrieves the value for the attribute, or sets an attribute
.last() will get the last element from the $("element").
.val() gets the value of information from input or sets the value
.addClass() , .removeClass(), .toggleClass() are the same as in vanilla JS.
.click("function") when element is clicked, a certain function is run
.keyPress("") use the "which" key for signaling a website. event.which === 13 is if the enter button is pressed.
.on() takes place of all the other methods, and we can enter our event in the .on("event type", function), this makes sure that you can do things with stuff that has not appeared on the page yet.
$(this) = jquery version of vanilla "this"

Effects and animations are in jQuery as well.
